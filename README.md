# WhatsApp Store Bot

Bot WhatsApp untuk toko online dengan fitur jualan menggunakan Baileys library.

## Fitur
- ✅ Menu produk dengan kategori
- ✅ Keranjang belanja
- ✅ Checkout dengan alamat pengiriman
- ✅ Multiple payment methods
- ✅ Tracking pesanan
- ✅ Pencarian produk
- ✅ Respons tombol dan list
